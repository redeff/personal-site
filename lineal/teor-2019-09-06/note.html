Proyectores

Una transformación lineal f:𝕍→𝕍f: \mathbb{V}\to \mathbb{V} se duce proyector sii f2=ff^2 = f.

⊕\oplus son proyectores

Dados dos ev en suma directa S⊕T=𝕍S\oplus T = \mathbb{V}, luego para cada v∈𝕍v \in \mathbb{V} existe una única escritura v=vt+vsv = v_t + v_s, donde vs∈S,vt∈Tv_s \in S, v_t \in T.

Si tomamos fv=s:s+t=v,s∈S,t∈Tf v = s : s + t = v, s \in S, t \in T.

Notemos que ff es lineal, es trivial.

Notemos además que:

  • S⊆ImfS \subseteq\mathop{\mathrm{Im}}f ya que para s⊆Ss \subseteq S tenemos fs=sfs = s

  • Imf⊆S\mathop{\mathrm{Im}}f \subseteq S ya que si fvfv está, por definición en SS.

  • Nuf⊆T\mathop{\mathrm{Nu}}f \subseteq T ya que si t∈Tt \in T, ft=0f t = 0.

  • T⊆NuFT \subseteq\mathop{\mathrm{Nu}}F por definición.

Es proyector ya que f(s+t)=fs+ft=sf (s+t) = fs + ft = s, y f2(s+t)=f(f(s+t))=fs=sf^2 (s+t) = f(f(s+t)) = fs = s.

Esta proyección se denomina proyección sobre el subespacio S, y se denota fSf_{S}.

Proyecciones son ⊕\oplus

Dado un proyector f:𝕍→𝕍f : \mathbb{V}\to \mathbb{V}, supongamos que v∈Nuf∩Imfv \in \mathop{\mathrm{Nu}}f \cap \mathop{\mathrm{Im}}f. Tenemos fv=0f v = 0, y a su vez fx=vf x = v para algún xx.

Notemos que v=fx=f2x=f(fx)=fv=0v = fx = f^2 x = f (fx) = f v = 0, luego v=0v = 0. Entonces la intersección entre Nuf∩Imf={0}\mathop{\mathrm{Nu}}f \cap \mathop{\mathrm{Im}}f = \{0\}.

Notemos además que Nuf+Imf=𝕍\mathop{\mathrm{Nu}}f + \mathop{\mathrm{Im}}f = \mathbb{V}, ya que podemos escribir v=(v−fv)+fvv = (v - fv) + fv, y f(v−fv)=fv−f(fv)=fv−fv=0f (v-fv) = fv - f(fv) = fv - fv = 0, osea que v−fv∈Nufv - fv \in \mathop{\mathrm{Nu}}f, y además fvfv claramente está en la imagen.

Luego tenemos que Nuf⊕Imf=𝕍\mathop{\mathrm{Nu}}f \oplus \mathop{\mathrm{Im}}f = \mathbb{V}.

Identidad como suma

Se tiene Id𝕍=fS+fT\mathop{\mathrm{Id}}_{\mathbb{V}} = f_{S} + f_{T} para cualquiera 𝕍=S+T\mathbb{V}= S + T.

Notemos además que si f:𝕍→𝕍f: \mathbb{V}\to \mathbb{V} proyector, se tiene (id−f)∘(id−f)=id2−id∘f−f∘id+f2=id−f−f+f=id−f(\mathop{\mathrm{id}}- f) \circ (\mathop{\mathrm{id}} - f) = \mathop{\mathrm{id}}^2 - \mathop{\mathrm{id}}\circ f - f \circ \mathop{\mathrm{id}}+ f^2 = \mathop{\mathrm{id}}- f - f + f = \mathop{\mathrm{id}}- f.

Espacio Hom

Dados 𝕍\mathbb{V}, 𝕎\mathbb{W} 𝕂\mathbb{K}-espacios vectoriales, se denomina: Hom𝕂(𝕍,𝕎)={tlf:𝕍→𝕎}\mathop{\mathrm{Hom}}_\mathbb{K}(\mathbb{V}, \mathbb{W}) = \{\mathop{\mathrm{tl}}f : \mathbb{V}\to \mathbb{W}\} Al conjunto de las transformaciones lineales entre 𝕍\mathbb{V} y 𝕎\mathbb{W}.

Se tiene que Hom𝕂(𝕍,𝕎)\mathop{\mathrm{Hom}}_\mathbb{K}(\mathbb{V}, \mathbb{W}) es un 𝕂\mathbb{K}-espacio vectorial.

Dimensión del Espacio Hom

Sea dim𝕍=n\dim \mathbb{V}= n y dim𝕎=m\dim \mathbb{W}= m, luego vamos a ver que dimHom(𝕍,𝕎)=n⋅m\dim \mathop{\mathrm{Hom}}(\mathbb{V}, \mathbb{W}) = n \cdot m

Isomorfismo Hom𝕂(𝕍,𝕎)∼𝕂dim𝕎×dim𝕍\mathop{\mathrm{Hom}}_\mathbb{K}(\mathbb{V}, \mathbb{W}) \sim \mathbb{K}^{\dim \mathbb{W}\times \dim \mathbb{V}}

Tomemos una base B=vi<nB = v_{i < n} de 𝕍\mathbb{V} y sea B′=wi<mB' = w_{i < m} una base de 𝕎\mathbb{W}. Para cada tlf:𝕍→𝕎\mathop{\mathrm{tl}}f : \mathbb{V}\to \mathbb{W}, asociémosle una matriz en 𝕂m×n\mathbb{K}^{m \times n}.

Para cada i<ni < n, sean λji\lambda_{ji} tal que fvi=∑λjiwjfv_i = \sum\limits\lambda_ji w_j. Contruyámonos entonces la matriz que se nota [f]BB′=(λ11λ12…λ1nλ21λ22…λ2n⋮⋮⋱⋮λm1λm2…λmn)[f]_{BB'} = \begin{pmatrix} \lambda_{11} & \lambda_{12} & \dots & \lambda_{1n} \\ \lambda_{21} & \lambda_{22} & \dots & \lambda_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ \lambda_{m1} & \lambda_{m2} & \dots & \lambda_{mn} \\ \end{pmatrix}

Proyectores como Matrices

Si tenemos f=fSf = f_{S} con 𝕍=S+T\mathbb{V}= S + T, y una base de 𝕍\mathbb{V} dada por B𝕍=BS∩BTB_\mathbb{V}= B_S \cap B_T con n=|BS|n = |B_S|, luego veamos que: [f]BB=(Idn×n000)[f]_{BB} = \begin{pmatrix} \mathop{\mathrm{Id}}_{n \times n} & 0 \\ 0 & 0 \\ \end{pmatrix}

Prop

Dados 𝕍\mathbb{V}, 𝕎\mathbb{W} una tlf:Hom(𝕍,𝕎)\mathop{\mathrm{tl}}f : \mathop{\mathrm{Hom}}(\mathbb{V}, \mathbb{W}) y bases BB de 𝕍\mathbb{V} y B′B' de 𝕎\mathbb{W}, se tiene que [fv]B′=[f]BB′×[v]B[fv]_{B'} = [f]_{BB'} \times [v]_B

Isomorfismo Hom𝕂(𝕍,𝕎)∼𝕂dim𝕎×dim𝕍\mathop{\mathrm{Hom}}_\mathbb{K}(\mathbb{V}, \mathbb{W}) \sim \mathbb{K}^{\dim \mathbb{W}\times \dim \mathbb{V}}

Definamos una ϕ:Hom(𝕍,𝕎)→𝕂m×n,ϕ=f↦[f]BB′\phi : Hom(\mathbb{V}, \mathbb{W}) \to \mathbb{K}^{m \times n}, \phi = f \mapsto [f]_{BB'}.

Es fácil ver que dadas tlf,g:𝕍→𝕎\mathop{\mathrm{tl}}f, g : \mathbb{V}\to \mathbb{W}, tenemos que ϕ(f+g)=ϕf+ϕg\phi (f + g) = \phi f + \phi g, es decir, [f+g]=[f]+[g][f+g] = [f] + [g]. Y que dado λ\lambda, se tiene λ⋅[f]=[λ⋅f]\lambda \cdot [f] = [\lambda \cdot f]

Para ver que es iso, definamos la inversa ψ:𝕂m×n→Hom(𝕍,𝕎),[ψMvi]B′=M*i\psi : \mathbb{K}^{m \times n} \to \mathop{\mathrm{Hom}}(\mathbb{V}, \mathbb{W}), [\psi M v_i]_{B'} = M_{*i}.

Notemos que (ϕ∘ψ)M=[ψM]BB′=M(\phi \circ \psi) M = [\psi M]_{BB'}= M, y además (ψ∘ϕ)f=f(\psi \circ \phi) f = f.

Corolario

dimHom(𝕍,𝕎)=dim𝕂m×n=m⋅n\dim \mathop{\mathrm{Hom}}(\mathbb{V}, \mathbb{W}) = \dim \mathbb{K}^{m \times n} = m \cdot n
