Expresar en Base

Dado v∈𝕍v \in \mathbb{V}, y una base ℬ𝕍\mathcal{B}_\mathbb{V}, tenemos que (v)ℬ𝕍=(λi)⇔∑ℬ𝕍i⋅λi=v(v)_{\mathcal{B}_\mathbb{V}} = (\lambda_i) \iff \sum\limits\mathcal{B}_{\mathbb{V}_i} \cdot \lambda_i = v

Igualdad

Si tenemos f,g∈Hom𝕍𝕎f, g \in \mathop{\mathrm{Hom}}\mathbb{V}\; \mathbb{W}, luego f=g⇔[f]ℬ𝕍ℬ𝕎=[g]ℬ𝕍ℬ𝕎f = g \iff [f]_{\mathcal{B}_\mathbb{V}\mathcal{B}_\mathbb{W}} = [g]_{\mathcal{B}_\mathbb{V}\mathcal{B}_\mathbb{W}}

Composición

Si f:𝕌→𝕍f : \mathbb{U}\to \mathbb{V} y g:𝕍→𝕎g : \mathbb{V}\to \mathbb{W}, luego [g∘f]=[g]×[f][g \circ f] = [g] \times [f]

Corolario

Tenemos 𝕍\mathbb{V}, 𝕎\mathbb{W} de dimensiones finitas respectivamente; y f∈Hom𝕍𝕎f \in \mathop{\mathrm{Hom}}\mathbb{V} \; \mathbb{W}. Luego fiso⇔[f]ℬ𝕍ℬ𝕎f \mathop{\mathrm{iso}}\iff [f]_{\mathcal{B}_\mathbb{V}\mathcal{B}_\mathbb{W}} es inversible.

Cambio de Base

Tomemos dos bases ℬ𝕍\mathcal{B}_\mathbb{V} y ℬ𝕍′\mathcal{B}_\mathbb{V}' de 𝕍\mathbb{V}, con dim𝕍=n\dim \mathbb{V}= n, y lo mismo para 𝕎\mathbb{W} de dimensión mm. Tenemos que: [f]ℬ′𝕍ℬ′𝕎=C(ℬ𝕎,ℬ′𝕎)×[f]ℬ𝕍ℬ𝕎×C(ℬ𝕍,ℬ′𝕍)−1[f]_{\mathcal{B}'_\mathbb{V}\mathcal{B}'_\mathbb{W}} = C(\mathcal{B}_\mathbb{W}, \mathcal{B}'_\mathbb{W}) \times [f]_{\mathcal{B}_\mathbb{V}\mathcal{B}_\mathbb{W}} \times C(\mathcal{B}_\mathbb{V}, \mathcal{B}'_\mathbb{V})^{-1}

Donde C(𝒜,ℬ)=[id]𝒜ℬC(\mathcal{A}, \mathcal{B}) = [\mathop{\mathrm{id}}]_{\mathcal{A}\mathcal{B}}.

Definición

Dados A,B∈𝕂n×nA, B \in \mathbb{K}^{n \times n}, decimos que A,BA, B son semejantes sii: A∼B⇔∃C∈GL𝕂n:A=C×B×C−1A \sim B \iff \exists C \in \mathop{\mathrm{GL}}\mathbb{K}^n : A = C \times B \times C^{-1}

Interpretación de Semejanza

Decimos que A∼B⇔∃f∈Hom𝕂n𝕂n:∃ℬ,ℬ′:[f]ℬ=A,[f]ℬ′=BA \sim B \iff \exists f \in \mathop{\mathrm{Hom}}\mathbb{K}^n \; \mathbb{K}^n :\exists \mathcal{B}, \mathcal{B}' : [f]_\mathcal{B}= A, [f]_{\mathcal{B}'} = B.

Rango de una Matriz

Si tenemos una matriz A∈𝕂n×mA \in \mathbb{K}^{n \times m}, y una ff tal que [f]=A[f] = A, luego decimos que: rgA=rg[f]=dimImf\mathop{\mathrm{rg}}A = \mathop{\mathrm{rg}}[f] = \dim \mathop{\mathrm{Im}}f

Dimensión de Soluciones de un Homogéneo

Si tenemos A∈𝕂m×nA \in \mathbb{K}^{m \times n}, luego tenemos dim{x:Ax=0}=n−rgA\dim \{x : Ax = 0\} = n - \mathop{\mathrm{rg}}A.

Sistemas no Homogéneos

Si tenemos el sistema lineal Ax=bAx = b, entonces tiene solución sii rgA=rg(A∣b)\mathop{\mathrm{rg}}A = \mathop{\mathrm{rg}} (A \mid b).

Notemos que rgA=rg(A∣b)⇔dimImA=dimIm(A|b)⇔dim⟨A*i⟩=dim(⟨A*i⟩+⟨b⟩)⇔\mathop{\mathrm{rg}}A = \mathop{\mathrm{rg}}(A \mid b) \iff \dim \mathop{\mathrm{Im}}A = \dim \mathop{\mathrm{Im}} (A | b) \iff \dim \langle A_{*i} \rangle = \dim (\langle A_{*i} \rangle + \langle b \rangle) \iff b es combinación lineal de las columnas de A⇔Ax=bA \iff Ax = b tiene solución.

Matrices Inversibles

Demostremos que dada A∈𝕂n×nA \in \mathbb{K}^{n \times n}, luego A×B=I⇒B×A=IA \times B = I \Rightarrow B \times A = I.

Tomemos f:𝕂n×n→𝕂n×n,fX=AXf : \mathbb{K}^{n \times n} \to \mathbb{K}^{n \times n}, fX = AX. Luego notemos que epif\mathop{\mathrm{epi}}f, ya que f(B×C)=A×(B×C)=(A×B)×C=Cf(B \times C) = A \times (B \times C) = (A \times B) \times C = C.

Luego por dimensión, tenemos isof\mathop{\mathrm{iso}}f. Pero además f(B×A−I)=A×(B×A−I)=A×B×A−A=I×A−A=0f(B \times A - I) = A \times (B \times A - I) = A \times B \times A - A = I \times A - A = 0, luego B×A=IB \times A = I
