Descomposición Primaria

Si tenemos f:𝕍→𝕍f : \mathbb{V}\to \mathbb{V}, y mf=∏pim_f = \prod p_i, con degpi>0\deg p_i > 0 coprimos dos a dos, luego:

  1. 𝕍=𝕍1+𝕍2+…+𝕍n\mathbb{V}= \mathbb{V}_1 + \mathbb{V}_2 + \dots + \mathbb{V}_n si 𝕍i=kerpi(f)\mathbb{V}_i = \ker p_i(f).

  2. 𝕍i\mathbb{V}_i es ff invariante.

  3. mf𝕍i=pim_{f_{\mathbb{V}_i}} = p_i

  4. Sea πi\pi_i una proyección sobre 𝕍i\mathbb{V}_i, luego πi=gi(f)\pi_i = g_i(f) para algún polinomio gg.

Es claro que mf𝕍i∣pim_{f_{\mathbb{V}_i}} \mid p_i, ya que por definición pi(f)|𝕍i=0p_i(f)|_{\mathbb{V}_i} = 0. Además, como son todos coprimos, y sabemos que 𝕍=∑𝕍i\mathbb{V}= \sum\limits\mathbb{V}_i, entonces: mf=∏mf𝕍im_f = \prod m_{f_{\mathbb{V}_i}} Pero entonces mf𝕍i=pim_{f_{\mathbb{V}_i}} = p_i

Notación

Una factorización mf=∏pim_f = \prod p_i con pip_i coprimo, se llama descomposición primaria, y el subespacio 𝕍i=kerpi(f)\mathbb{V}_i = \ker p_i(f) se llama componente pip_i-primaria.

Condición de Diagonalizabilidad

Una transformación lineal f:𝕍→𝕍f : \mathbb{V}\to \mathbb{V} es diagonalizable ⇔\iff mf=∏(x−ai)m_f = \prod (x-a_i) con todos coprimos.

(⇒\Rightarrow)

Sean {λi≤r}\{\lambda_{i \leq r}\} los autovalores sin repetición de ff. Además 𝕍=⨁i𝕍λi\mathbb{V}= \bigoplus_i \mathbb{V}_{\lambda_i}

Sabemos además que pi=mf𝕍λi=x−λip_i = m_{f_{\mathbb{V}_{\lambda_i}}} = x - \lambda_i. Luego mf=lcmpi=∏pi=∏(x−λi)m_f = \mathop{\mathrm{lcm}}p_i = \prod p_i = \prod (x - \lambda_i). Ya estamos.

(⇐\Leftarrow)

Supongamos que mf=∏(x−λi)m_f = \prod (x - \lambda_i), con λi≠λj\lambda_i \neq \lambda_j. Con lo anterior, sabmos que 𝕍=⨁𝕍i\mathbb{V}= \bigoplus \mathbb{V}_i con 𝕍i=kerpi(f)=ker(f−λif)=𝕍λi\mathbb{V}_i = \ker p_i(f) = \ker (f - \lambda_if) = \mathbb{V}_{\lambda_i} Luego 𝕍\mathbb{V} se escribe como suma de sus autoespacios. Ya estamos.

Nilpotentes

Índice de nilpotencia. Dada f:𝕍→𝕍f : \mathbb{V}\to \mathbb{V}, el índice de nilpotencia es el menor nn tal que fn=0f^n = 0.

Notemos que si ff es nilpotente, entonces mf=xim_f = x^i donde ii es el índice de nilpotencia de ff. Además χf=xn\chi_f = x^n.

Esto implica que la única nilpotente diagonalizable es la nula.

Caracterización de Nilpotentes

Si tenemos f:𝕍→𝕍f : \mathbb{V}\to \mathbb{V}, luego es equivalente:

  1. ff nilpotente

  2. fn=0f^n = 0

  3. mf=xkm_f = x^k

  4. χf=xn\chi_f = x^n

Son todas triviales.
