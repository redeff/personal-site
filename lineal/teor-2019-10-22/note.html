Plinomio minimal

Dada f:𝕍→𝕍f : \mathbb{V}\to \mathbb{V}, el polinomio minimal mfm_f es el polinomio mónico de menor grado, tal que mf(f)=0m_f(f) = 0 (la tl nula).

Por ejemplo, si dimV>0\dim V > 0, para f=idf = \mathop{\mathrm{id}}, tenemos que mf(x)=x−1m_f(x) = x-1 y para f=0f = 0 tenemos mf(x)=0m_f(x) = 0

Caso 𝕍={0}\mathbb{V}= \{0\}

Es ese caso, la única función f:𝕍→𝕍f : \mathbb{V}\to \mathbb{V} es la tl nula, que también es la identidad, luego mf(x)=1m_f(x) = 1, ya que id=0\mathop{\mathrm{id}}= 0

Se rompe para 𝕍\mathbb{V} infinito

Si tenemos f:k[x]→k[x]f : k[x] \to k[x], f(p(x))=xp(x)f(p(x)) = xp(x), luego tenmos que: Q(f)(p(x))=∑aixip(x)=Q(x)p(x)Q(f)(p(x)) = \sum\limits a_i x^i p(x) = Q(x)p(x) Luego ningún no nulo QQ multiplicado por ff da la tl nula.

Mínimo común múltiplo

Sea f:𝕍→𝕍f : \mathbb{V}\to \mathbb{V}, con 𝕎⊆𝕍\mathbb{W}\subseteq\mathbb{V} ff-invariante. Luego mf𝕎∣mfm_{f_\mathbb{W}} \mid m_f.

También tenemos

  1. mf=lcm{mfv:v∈𝕍}m_f = \mathop{\mathrm{lcm}}\{m_{f_v} : v \in \mathbb{V}\}

  2. Si 𝕍=𝕍1+…+𝕍2\mathbb{V}= \mathbb{V}_1 + \dots + \mathbb{V}_2, luego mf=lcm{mf𝕍i}im_f = \mathop{\mathrm{lcm}}\{m_{f_{\mathbb{V}_i}}\}_i

  3. Si 𝕍=∑⟨vi⟩f\mathbb{V}= \sum\limits\langle v_i \rangle_f, luego mf=lcm{mfvi}im_f = \mathop{\mathrm{lcm}}\{m_{f_{v_i}}\}_i

Notemos que 0=mf(f)(v)=mf(fv)(v)0 = m_f(f)(v) = m_f(f_v)(v), luego mf(fv)=0m_f(f_v) = 0, entonces mfv∣mfm_{f_v} \mid m_f. Escribamos entonces mf(x)=mfv(x)⋅qv(x)m_f(x) = m_{f_v}(x) \cdot q_v(x).

Vamos a ver que si existe p(x):∀w:mfw∣p(x)p(x) : \forall w : m_{f_w} \mid p(x), luego demostremos que mf∣pm_f \mid p.

Tenemos p(f)(v)=(p′v(f)⋅mfv(f))(v)=p′v(f)(mfv(f)(v))=p′v(f)(0)=0p(f)(v) = (p'_v(f) \cdot m_{f_v}(f)) (v) = p'_v(f) (m_{f_v}(f)(v)) = p'_v(f)(0) = 0, luego p(f)(v)=0p(f)(v) = 0 para todo vv, luego mf∣pm_f \mid p.
