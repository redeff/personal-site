Teorema

Para 𝕍\mathbb{V} finitamente ganerado. Si ⟨X⟩=𝕍\langle X \rangle = \mathbb{V} y Y⊆𝕍Y \subseteq \mathbb{V} es li, luego #X≥#Y\# X \geq \#Y.

Corolario

Todas las bases de un 𝕍\mathbb{V} f.g. tienen el mismo cardinal.

Notar que si un conjunto tiene una base finita, entonces todas sus bases son finitas.

Notar que dim𝕂𝕍=0⇔𝕍={0}\dim_\mathbb{K}\mathbb{V}= 0 \iff \mathbb{V}= \{0\}

Dimensión de Subespacios

Sea 𝕍\mathbb{V} tal que dim𝕍=n\dim \mathbb{V}= n. Sea 𝕌\mathbb{U} subesp. de 𝕍\mathbb{V}. Notemos que 𝕌\mathbb{U} tiene dimensión finita y dim𝕌≤dim𝕍\dim \mathbb{U}\leq \dim \mathbb{V}.

Notemos que si 𝕌\mathbb{U} fuese de dimensión infinita, podríamos encontrarnos un subconj. de 𝕌\mathbb{U} li de tamaño arbitrario. En particular de tamaño n+1n+1, entonces habría n+1n+1 vectores li en 𝕍\mathbb{V}, por qlo que no puede tener dimensión dim𝕍=n\dim \mathbb{V}= n.

Luego 𝕌\mathbb{U} tiene una base finita XX. Tendremos que XX es li., y además X⊆𝕍X \subseteq \mathbb{V}, pero luego necesariamente #X≤dim𝕍\# X \leq \dim \mathbb{V} (pq son li), entonces ya está.

Caracterizando las Bases

Sea dim𝕍=n\dim \mathbb{V}= n y sea X⊆𝕍∣#X=nX \subseteq \mathbb{V}\mid \#X = n. demostremos que: Xli⇔⟨X⟩=𝕍⇔XbaseX \;\text{li} \iff \langle X \rangle = \mathbb{V}\iff X \;\text{base}

Supongamos que XX es li. Luego notemos que XX es base de ⟨X⟩\langle X \rangle, luego dim⟨X⟩=n\dim \langle X \rangle = n, luego ⟨X⟩=𝕍\langle X \rangle = \mathbb{V}.

Supongamos que ⟨X⟩=𝕍\langle X \rangle = \mathbb{V}. Notemos que para cualquier Y⊊XY \subsetneq X tenemos #Y<n\# Y < n, luego ⟨Y⟩≠𝕍\langle Y \rangle \neq \mathbb{V}, ya que osinó dim𝕍≤n−1\dim \mathbb{V}\leq n-1. Entonces XX es generador minimal.

Dado 𝕍\mathbb{V} de dimensión dim𝕍=n\dim \mathbb{V}= n. Sea X⊆𝕍X\subseteq\mathbb{V} li, luego existe X⊆B⊆𝕍X \subseteq B \subseteq\mathbb{V} tal que BB base de 𝕍\mathbb{V}.

Si #Y=n\# Y = n ya estamos. Notemos que YY es un conjunto li maximal. Luego es base de 𝕍\mathbb{V}, luego #Y=n\# Y = n, ya estamos.

Suma de Subespacios

Dado S,T⊆𝕍S, T \subseteq\mathbb{V} subespacios, se define: S+T={s+t∣s∈S,t∈T}S + T = \{s + t \mid s \in S, t\in T\}

Demostremos S+TS + T subespacio. Notemos

  • S+T≠∅S + T \neq \emptyset

  • dados v,w∈S+Tv, w \in S + T tomemos s+t=vs + t = v, s′+t′=ws' + t' = w, luego v+w=(s+s′)+(t+t′)∈S+Tv + w = (s + s') + (t + t') \in S+T

  • dado v∈S+Tv \in S + T y k∈𝕂k \in \mathbb{K}, notemos v=s+tv = s + t, luego k⋅v=k⋅t+k⋅s∈S+Tk \cdot v = k \cdot t + k \cdot s \in S + T.

Demostremos S+T=⟨S∪T⟩S + T = \langle S \cup T\rangle. Tenemos claramente S∪T⊆S+TS \cup T \subseteq S + T, además como es subespacio, tenemos ⟨S∪T⟩⊆S+T\langle S \cup T \rangle \subseteq S + T.

Además claramente todo v∈S+Tv \in S + T es una combinación lineal de cosas de S∪TS \cup T, luego S+T⊆⟨S∪T⟩S + T \subseteq\langle S \cup T \rangle, y ya estamos.

Teorema de la Dimensión

Dados S,TS, T subespacios de 𝕍\mathbb{V} con dim𝕍=n\dim \mathbb{V}= n, se tiene: dimS+T=dimS+dimT−dimS∩T\dim S + T = \dim S + \dim T - \dim S \cap T

Sea BB base de S∩TS \cap T. Tomemos B⊆BTB \subseteq B_T base de TT y B⊆BSB \subseteq B_S base de SS. Tomemos BS+T=BS∪BTB_{S + T} = B_S \cup B_T.

Notemos que BS∩BTB_S \cap B_T es BB, ya que si hubiera v∈BS∩BT−Bv \in B_S \cap B_T - B, tendríamos que BB no es li maximal (vv es li con BB).
