Anulador

Dado SS un subespacio de 𝕍\mathbb{V}, si definimos T={x:𝕍:∀ϕ∈S0:ϕx=0}T = \{x : \mathbb{V}: \forall \phi \in S^0 : \phi x = 0\}, luego S=TS = T.

Es claro que S⊆TS \subseteq T. Vemos que T⊆ST \subseteq S. Sea x∈Tx \in T, x∉Sx \notin S, tomemos vv una base de 𝕍\mathbb{V}, con vi<rv_{i < r} base de SS, y vr=xv_r = x. Sea la base dual ϕ=v*\phi = v^*. Como ϕi≥rvi<r=0\phi_{i \geq r}v_{i < r} = 0, tenemos que ϕi≥r\phi_{i \geq r} es base de S0S^0.

Luego ϕr⊆S0\phi_r \subseteq S^0, entonces ϕrx=0\phi_r x = 0, pero x=vrx = v_r, pero entonces por definición de base dual tenemos ϕrx=1\phi_r x = 1. Contradicción.

Observación

Dado 𝕍\mathbb{V} finitamente generado y SS sub de 𝕍\mathbb{V}, lugo si ϕ\phi es base de S0S^0, luego S={x∈𝕍:∀i:ϕix=0}S = \{x \in \mathbb{V}: \forall i : \phi_ix = 0\}.
