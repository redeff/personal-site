Espacio Dual

Dado 𝕍\mathbb{V} un 𝕂\mathbb{K}-ev, llamo el dual de 𝕍\mathbb{V} como 𝕍*=Hom𝕂(𝕍,𝕂)\mathbb{V}^* = \mathop{\mathrm{Hom}}_\mathbb{K}(\mathbb{V},\mathbb{K}). Si dim𝕍=n\dim \mathbb{V}= n, luego tenemos dim𝕍*=n⋅1=n\dim \mathbb{V}^* = n \cdot 1 = n.

A los elementos f∈𝕍*f \in \mathbb{V}^* se los llama funcionales lineales.

Base de 𝕍*\mathbb{V}^*

Dada una base ℬ\mathcal{B}, definimos una familia de nn transformaciones lineales indejadas por 1≤j≤n1 \leq j \leq n, definidas por: ϕj=ℬi↦{1Si i=j0osino\phi_j = \mathcal{B}_i \mapsto \begin{cases} 1 & \text{Si $i = j$} \\ 0 & \text{osino} \end{cases}.

Vamos a probar que ϕ\phi es base, y vamos a llamarla la base dual de ℬ\mathcal{B}, o ℬ*\mathcal{B}^*.

Demo: Como dim𝕍*=n\dim \mathbb{V}^* = n, alcanza con probar que los ϕ\phi son li\mathop{\mathrm{li}}.

Supongamos que 0=∑λiϕi0 = \sum\limits\lambda_i\phi_i, luego evaluamos esta función nula en la base, y tenemos, para cada vector de la base ℬi\mathcal{B}_i, se cumple

Expresar elementos de 𝕍*\mathbb{V}^* en una base

Dado un ψ∈𝕍*\psi \in \mathbb{V}^*. Se puede expresar como ψ=∑λi⋅ℬi*\psi = \sum\limits\lambda_i \cdot \mathcal{B}^*_i Para cada ℬj\mathcal{B}_j, tomemos ψ=∑λi⋅ℬi*\psi = \sum\limits\lambda_i \cdot \mathcal{B}^*_i ψℬj=∑λi⋅ℬi*ℬj\psi\mathcal{B}_j = \sum\limits\lambda_i \cdot \mathcal{B}^*_i \mathcal{B}_j ψℬj=λj⋅ℬj*ℬj\psi\mathcal{B}_j = \lambda_j \cdot \mathcal{B}^*_j \mathcal{B}_j ψℬj=λj\psi\mathcal{B}_j = \lambda_j Luego los coeficientes en la base dual, son las imágenes de la base original.
