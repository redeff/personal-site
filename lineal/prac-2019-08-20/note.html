Rec

B⊆𝕍B \subseteq \mathbb{V} es base ⇔\iff BB es li y ⟨B⟩=𝕍\langle B \rangle = \mathbb{V}

Ejercicio 1

Dado 𝕍=𝕂n[x]={p∈𝕂[x]∣p=0 ó deg(p)≤n}\mathbb{V}= \mathbb{K}_n[x] = \{p \in \mathbb{K}[x] \mid p = 0 \text{ ó } \deg(p) \leq n\}.

Si B={p∣deg(p)=i|0≤i≤n}B = \{p \mid \deg(p) = i | 0 \leq i \leq n\}, demostrar que BB es base.

La matriz en E={xi∣0≤i≤n}E = \{x^i \mid 0 \leq i \leq n\} es triangular.

Demo 2

Inducción. n=0n = 0 es trivial.

Si n≥1n \geq 1, supongamos ∑λiBi=0\sum\limits\lambda_iB_i = 0. El coeficiente nn de ese polinomio tiene que ser cero, luego λn=0\lambda_n = 0, luego 0=∑i<nλiBi=00 = \sum\limits_{i < n} \lambda_iB_i = 0, pero esto no puede pasar por la hipótesis inductiva.

Ejercicio 8

Tomemos S={A∈𝕂n×n|∑jAij=0=∑iAij}S = \left\{A \in \mathbb{K}^{n \times n} \; \middle| \; \sum\limits_j A_{ij} = 0 = \sum\limits_i A_{ij}\right\}

Calcular dimS\dim S. Notemos que si dejamos libre los AijA_{ij} con i<ni < n y j<nj < n y tomamos Anj=−∑i<nAijA_{nj} = -\sum\limits_{i < n} A_{ij}, y Ain=−∑j<nAijA_{in} = -\sum\limits_{j < n} A_{ij}, y Ann=−∑i,j<nAijA_{nn} = -\sum\limits_{i,j < n} A_{ij}, esta matriz cumple, y son todas las que cumplen.

Luego da dimS=(n−1)2\dim S = (n-1)^2

Intersección y suma de subespacios

Dados SS, TT sub. de 𝕍\mathbb{V}, se tiene

  • S∩T=S∩TS \cap T = S \cap T

  • S+T={s+t∣s∈S,t∈T}S + T = \{s + t \mid s \in S, t \in T\}

  • S⊕T=S+TS \oplus T = S + T si están en suma directa, que se da cuando:

    • S∩T=0⇔S \cap T = {0} \iff

    • dim(S+T)=dimS+dimT⇔\dim (S + T) = \dim S + \dim T \iff

    • ∀v∈S+T∃!s∈S,t∈T∣s+t=v\forall v \in S + T \; \exists! \; s \in S, t\in T \mid s + t = v

Complemento

Cuando S⊕T=𝕍S \oplus T = \mathbb{V}, decimos que SS es complemento de TT. Se sabe que eso implica dimS=n−dimT\dim S = n - \dim T, es decir la dimensión de los complementos es toda la misma
