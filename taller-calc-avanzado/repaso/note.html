Axiomas

Monoide Conmutativo

  1. (a*b)*c=a*(b*c)(a * b) * c = a * (b*c)

  2. a*b=b*aa*b = b*a

  3. ∃e:e*a=a\exists e : e*a = a

Propiedades

  1. ee es único

  2. Inversos, si existen, son únicos.

  3. (a*b)′=a′*b′(a*b)' = a'*b', si existen.

Morfismos

Un morfismo de monoides f:A→Bf : A \to B es aquel que cumple

  1. f(a*b)=f(a)*f(b)f(a * b) = f(a) * f(b)

  2. f(e)=ef(e) = e

Submonoides

Es un subconjunto de B∈AB \in A que contiene al neutro y es cerrado bajo .

Grupo de Unidades

El conjunto de los elementos inversibles U(A)∈AU(A) \in A es un submonoide.

Grupo

Es un monoide (conmutativo) donde todos sus elementos son inversibles.

Anillo

Es un conjunto AA con dos operaciones, ++ y ⋅\cdot, tal que

  1. (A,+)(A, +) es un grupo conmutativo.

  2. (A,⋅)(A, \cdot) es un monoide conmutativo.

  3. a(b+c)=ab+aca(b + c) = ab + ac

Propiedades

  1. a+c=b+c⇒a=ba + c = b + c \Rightarrow a = b

  2. a0=0a0 = 0

  3. a(−b)=−aba(-b) = -ab

  4. (−a)(−b)=ab(-a)(-b) = ab

  5. Si 1=01 = 0, luego A={0}A = \{0\}

Un divisor de 0 es un aa tal que existe un b∈A\{0}b \in A \setminus \{0\} con ab=0ab = 0.

Dominio

Un anillo es dominio cuando no tiene divisores de 00 no nulos

Cuerpo

Un anillo es cuerpo cuando todos los elementos no nulos son inversibles, es decir U(A)=A\{0}U(A) = A \setminus \{0\}

  1. a<b⇒a+c<b+ca < b \Rightarrow a + c < b + c (Compatibilidad con la suma)

  2. a<b,0<c⇒ac<bca < b, 0 < c \Rightarrow ac < bc (Compatibilidad con el producto)
