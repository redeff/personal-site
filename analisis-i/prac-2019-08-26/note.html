Ejercicio 1

Dado el conjunto A={(x,y)∈ℝ2:−x<y,y+x2−2≤0}A = \{(x, y) \in \mathbb{R}^2 : -x < y, y + x^2 - 2 \leq 0\} No es abierto ya que el punto (0,2)(0, 2) está en AA pero (0,2+ε)∉A(0, 2 + \varepsilon) \notin A para todo ε>0\varepsilon> 0.

Notemos que tampoco es cerrado, ya que (0,0)∉A(0, 0) \notin A, pero (0,ε)∈A(0, \varepsilon) \in A para cualquier ε\varepsilon suficientemente chico.

Conjuntos Acotados

Un conjunto A⊆ℝnA \subseteq\mathbb{R}^n es acotado si existe alguna bola A⊆BM(0)A \subseteq B_M(0)

Propiedades

  • A∘⊆AA^\circ \subseteq A

  • A∘A^\circ es abierto

  • si AA abierto luego A∘=AA^\circ = A

Punto de Acumulación (Clausura)

Un punto x∈ℝnx \in \mathbb{R}^n es un punto de acumulación si existe una sucesión a:ℕ→Aa : \mathbb{N}\to A tal que limi→∞ai=x\lim\limits_{i \to \infty} ai = x.

Se define al conjunto de puntos de acumulación como clA=A¯\mathop{\mathrm{cl}}A = \overline A. Es claro que A⊆A¯A \subseteq\overline A, y además A¯=clA\overline A = \mathop{\mathrm{cl}}A es cerrado.

Borde

el borde de AA se define como ∂A=clA−A∘\partial A = \mathop{\mathrm{cl}}A - A^\circ.

Alternativamente, ∂A={x:∀ε>0:Br(x)∩A≠∅,Br(x)∩AC≠∅}\partial A = \{ x : \forall \varepsilon> 0 : B_r(x) \cap A \neq \emptyset, B_r(x) \cap A^C \neq \emptyset \}

Cónicas

Una cónica es una es una curva en ℝ2\mathbb{R}^2 que surge de cortar un cono con un plano. Se dividen en:

  • Circunferencia: x2+y2=r2x^2 + y^2 = r^2.

  • Elipse: x2a2+y2b2=1\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1

  • Parábola: y=x2y = x^2

  • Hipérbola: x2a2−x2b2=1\frac{x^2}{a^2} - \frac{x^2}{b^2} = 1
