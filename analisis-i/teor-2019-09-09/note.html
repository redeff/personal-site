Arcoconexo

Un conjunto C⊆ℝnC \subseteq\mathbb{R}^n es arcoconexo sii para cualquiera p,q∈Cp, q \in C, los puedo unir por una curva contínua dentro de CC. Es decir: ∀p,q∈C:∃αcont:[0;1]→C:α0=p,α1=q\forall p, q \in C : \exists \alpha \mathop{\mathrm{cont}} : [0; 1] \to C : \alpha 0 = p, \alpha 1 = q

Principio de Fermat

Dada una función f:[a,b]→ℝf : [a, b] \to \mathbb{R}, y algún c∈(a,b)c \in (a, b), con ff derivable en cc, si ff tiene un máximo o un mínimo local en cc entonces tenemos f′c=0f'c = 0.

Demo: Supongamos que hay un mínimo local en cc, luego se tiene que en un entorno de cc, se tiene fc<f(c+h)fc < f(c+h), pera todo h<δh < \delta para algún δ\delta.

Luego tenemos:

limh→0+fc−f(c+h)h≥0\lim\limits_{h \to 0^+} \frac{fc - f(c+h)}{h} \geq 0, y limh→0−fc−f(c+h)h≤0\lim\limits_{h \to 0^-} \frac{fc - f(c+h)}{h} \leq 0, luego si el límite existe, debe ser 00.
