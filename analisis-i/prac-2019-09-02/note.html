Límite

Dado una función f:A→ℝf : A \to \mathbb{R}, con A⊆ℝA \subseteq\mathbb{R}, luego para todo x0∈Ax_0 \in A decimos que: limx→x0fx=ℓ⇔∀ε>0:∃δ>0:∀x∈A:|x−x0|<δ⇒|fx−fx0|<ε\lim\limits_{x \to x_0} fx = \ell \iff \forall \varepsilon>0 : \exists \delta > 0 : \forall x \in A : |x - x_0| < \delta \Rightarrow|fx - fx_0| < \varepsilon

Ejemplos

Tomemos fx=x2fx = x^2, y demostremos limx→2fx=4\lim\limits_{x \to 2} fx = 4. tenemos |x2−4|=|x−2|⋅|x+2||x^2-4| = |x-2| \cdot |x+2|. Por desigualdad triangular tenemos |x+2|<δ+4|x+2| < \delta + 4, y tenemos |x−2|<δ|x-2| < \delta, luego tenemos: |x2−4|<(δ+4)⋅δ|x^2-4| < (\delta + 4) \cdot \delta, entonces queremos un δ\delta tal que (δ+4)⋅δ<ε(\delta + 4) \cdot \delta < \varepsilon.

Si le pedimos que δ<1\delta < 1, tomemos δ<min{ε1+4,1}<εδ+4\delta < \min \left\{\frac{\varepsilon}{1 + 4}, 1 \right\}< \frac{\varepsilon}{\delta + 4}, y ya estamos.

Alternativamente, usamos que δ2<δ\delta^2 < \delta para δ<1\delta < 1, entonces tenemos (δ+4)⋅δ=δ2+4⋅δ<5⋅δ(\delta + 4) \cdot \delta = \delta^2 + 4 \cdot \delta < 5 \cdot \delta, luego tomamos δ<min{ε5,1}\delta < \min \left\{\frac{\varepsilon}{5}, 1\right\} y estamos.

Cero por Acotado

Si tenemos limx→x0fx=0\lim\limits_{x \to x_0} fx = 0 y gg acotada, luego limx→x0gx⋅fx=0\lim\limits_{x \to x_0} gx \cdot fx = 0.

L’Hôpital

Si ff y gg con continuas y derivables en un entorno de c∈ℝc \in \mathbb{R}, y tenemos fc=gc=0f c = g c = 0 y g′c≠0g' c \neq 0. Luego si existe el límite ℓ=limx→cf′xg′c\ell = \lim\limits_{x \to c} \frac{f' x}{g' c}, luego limx→cfcgc=ℓ\lim\limits_{x \to c} \frac{f c}{g c} = \ell.

Def de Wikipedia

Dadas funciones derivables ff y gg definidas en un entorno punteado A⊆ℝA \subseteq\mathbb{R} de a∈ℝa \in \mathbb{R}, si limx→afx=limx→agx=0\lim\limits_{x \to a} fx = \lim\limits_{x \to a} gx = 0 y gx≠0∀x∈Ag x \neq 0 \; \forall x \in A, y se tiene ℓ=limx→af′xg′x\ell = \lim\limits_{x \to a} \frac{f' x }{g' x }, luego ℓ=limx→afxgx\ell = \lim\limits_{x \to a} \frac{fx }{gx }

Límite Multivariable

Dado una función f:A→ℝmf : A \to \mathbb{R}^m, con A⊆ℝnA \subseteq\mathbb{R}^n, luego para todo x0∈A¯x_0 \in \overline{A} decimos que: limx→x0fx=ℓ⇔∀ε>0:∃δ>0:∀x∈A:||x−x0||<δ⇒||fx−fx0||<ε\lim\limits_{x \to x_0} fx = \ell \iff \forall \varepsilon>0 : \exists \delta > 0 : \forall x \in A : ||x - x_0|| < \delta \Rightarrow||fx - fx_0|| < \varepsilon

Ejemplos

Demostremos lim(x,y)→(−1,3)2x+y=1\lim\limits_{(x, y) \to (-1, 3)} 2x + y = 1. Tenemos que encontrar δ\delta tal que |2x+y−1|<ε|2x + y - 1| < \varepsilon.

Tenemos |x+1|<δ|x+1| < \delta y |y−3|<δ|y-3| < \delta. |2x+2|<2⋅δ|2x+2| <2 \cdot \delta. Sumando las dos últimas por desigualdad triangular tenemos:

|2x+y−1|<3⋅δ|2x + y - 1| < 3 \cdot \delta, entonces tomamos δ=e3\delta = \frac{e}{3}, y ya estamos.
