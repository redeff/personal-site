Definición

Una función f:A→Bf : A \to B es una relación f⊆A×Bf \subseteq A \times B, donde AA se llama dominio y BB codominio que cumple ∀a∈A∃!b:(a,b)∈f\forall a \in A \; \exists ! b : (a, b) \in f, que se nota fa=bfa = b.

Imagen

Dado C⊆AC \subseteq A, definimos fC:={fc:c∈C}f C := \{f c : c \in C\}. Al conjunto fAfA se lo llama imagen de ff (ℑf\Im f).

Preimagen

Dado D⊆BD \subseteq B, definimos f−1D={a∈A:fa∈D}f^{-1}D = \{a \in A : fa \in D \}.

Restricción

Dado C⊆AC \subseteq A, definimos f|C:C→B=f∩(C×B)f |_C : C \to B = f \cap (C \times B).

Composición

Dadas f:A→Bf : A \to B y g:B→Cg : B \to C, definimos g∘f:A→Cg \circ f : A \to C como (g∘f)x=g(fx)(g \circ f) x = g (fx).

Inyectividad

Si ∀a,b∈Domf:fa=fb⇒a=b\forall a, b \in \text{Dom} f : fa = fb \Rightarrow a = b.

Prop

f:A→Bf : A \to B es inyectiva ⇔∃g:B→A:g∘f=idA\iff \exists g : B \to A : g \circ f = \text{id}_A.

Sobreyectividad

ff es sobreyectiva si fA=Bf A = B.

Prop

f:A→Bf : A \to B es sobreyectida ⇔∃g:B→A:f∘g=idB\iff \exists g : B \to A : f \circ g = \text{id}_B.
