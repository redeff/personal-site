Par Ordenado

El par ordenado se nota (a,b)(a, b) y tiene la propiedad de que (a,b)=(c,d)⇔a=c y b=d(a, b) = (c, d) \iff a=c \text{ y } b = d

El par (a,b)(a, b) se define como {{a},{a,b}}\{\{a\}, \{a, b\}\}, que notemos que cumple la prop del párrafo anterior

Notemos que si a=ba = b luego |(a,b)|=1|(a, b)| = 1, osino |(a,b)|=2|(a, b)| = 2

Supongamos {{a},{a,b}}={{c},{c,d}}\{\{a\}, \{a, b\}\} = \{\{c\}, \{c, d\}\}, luego a=ba = b iff c=dc = d para que tengan la misma cantidad de elementos.

Si son iguales notemos que tenemos {{a}}={{c}}\{\{a\}\} = \{\{c\}\}, entonces a=ca = c, y ya estamos

Si son aa y bb distintos, luego ambos pares ordenados tienen tamaño 22. Además para que {{a},{a,b}}={{c},{c,d}}\{\{a\}, \{a, b\}\} = \{\{c\}, \{c, d\}\}, necesariamente {a,b}={c,d}\{a, b\} = \{c, d\} y {a}={c}\{a\} = \{c\}, ya que para que dos sean iguales deben tener el mismo tamaño

Entonces a=ca = c, y {a,b}={c,d}\{a, b\} = \{c, d\}, entonces b=db = d y estamos
