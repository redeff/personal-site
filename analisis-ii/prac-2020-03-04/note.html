Sistemas de Ecuaciones Lineales Homogéneos

Dado el sistema X′(t)=A(t)X(t)X'(t) = A(t)X(t) Admite una solución general de la forma X(t)=λ1X1(t)+…+λnXn(t)X(t) = \lambda_1X_1(t) + \dots + \lambda_nX_n(t) Donde {Xi}i\{X_i\}_i es una base del espacio solución.

Se dice que la matriz fundamental del sistema es Q(t)=[X1(t)|…|Xn(t)]Q(t) = [X_1(t) | \dots | X_n(t)] Y como son li, se tiene detQ≠0\det Q \neq 0.

Sistemas de Ecuaciones Lineales Heterogéneos

Tenemos un sistema de la pinta X′(t)=A(t)X(t)+F(t)X'(t) = A(t)X(t) + F(t) Y vamos a ver el caso además donde A(t)=AA(t) = A es una matriz constante.

Caso AA diagonalizable

Tenemos CAC−1=DCAC^{-1} = D Si introducimos la variable X(t)=C−1Y(t)X(t) = C^{-1}Y(t), nos queda (C−1Y(t))′=AC−1Y(t)+F(t)(C^{-1}Y(t))' = AC^{-1}Y(t) + F(t) C−1Y′(t)=AC−1Y(t)+F(t)C^{-1}Y'(t) = AC^{-1}Y(t) + F(t) Y′(t)=DY(t)+CF(t)Y'(t) = DY(t) + CF(t) Si tomamos G(t)=CF(t)G(t) = CF(t), luego sabiendo que DD es diagonal queda {y′1(t)=λ1yt(t)+g1(t)y′2(t)=λ2y2(t)+g2(t)⋮y′n(t)=λnyn(t)+gn(t)\begin{cases} y'_1(t) = \lambda_1y_t(t) + g_1(t) \\ y'_2(t) = \lambda_2y_2(t) + g_2(t) \\ \vdots \\ y'_n(t) = \lambda_ny_n(t) + g_n(t) \\ \end{cases} Que las puedo resolver independientemente.

Caso no Diagonalizable

Tenemos X(t)=AX(t)+F(t)X(t) = AX(t) + F(t) La solución viene dada por X(t)=Xh(t)+Xp(t)X(t) = X_h(t) + X_p(t) Donde XhX_h es una sol gral al sist. homogeneo, y XpX_p es una particular al sistema no homogeneo.

Tenemos que Xh(t)=Q(t)CX_h(t) = Q(t) C, donde C=[c1c2⋮cn].C = \begin{bmatrix} c_1 \\ c_2 \\ \vdots \\ c_n \end{bmatrix}. son los coeficientes.

Por el método de variación de constantes suponemos Xp(t)=Q(t)C(t)X_p(t) = Q(t)C(t), luego reemplazando en el sistema tenemos Q′(t)C(t)+Q(t)C′(t)=AQ(t)C(t)+F(t)Q'(t)C(t) + Q(t)C'(t) = AQ(t)C(t) + F(t) Q′C+QC′=AQC+FQ'C + QC' = AQC + F (Q′−AQ)C+QC′=F(Q' - AQ)C + QC' = F QC′=FQC' = F C′=Q−1FC' = Q^{-1}F Luego integramos y estamos.

Notemos que los autovalores son λ1=−1\lambda_1 = -1 y λ2=1\lambda_2 = 1, y tenemos Sλ1=⟨1,3⟩S_{\lambda_1} = \langle 1, 3\rangle, y Sλ2=⟨1,1⟩S_{\lambda_2} = \langle 1, 1 \rangle
